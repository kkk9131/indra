# X (Twitter) アルゴリズム ガイド

## 概要

- 目的: アルゴリズムに最適化されたポスト作成の参考資料
- ソース: github.com/twitter/the-algorithm + 2026年最新情報

## 1. スコアリングシステム

### 正の信号（加点要素）

| 信号                | 重要度 | 説明                                  |
| ------------------- | ------ | ------------------------------------- |
| リプライ            | 最高   | ReplyScore, ReplyEngagedByAuthorScore |
| いいね              | 高     | FavoriteScore                         |
| リツイート          | 高     | RetweetScore                          |
| 滞在時間            | 中     | DwellScore, GoodClickV1/V2Score       |
| 動画視聴            | 中     | VideoQualityViewScore                 |
| ブックマーク/シェア | 低〜中 | BookmarkScore, ShareScore             |

### 負の信号（減点要素）

| 信号                     | ペナルティ |
| ------------------------ | ---------- |
| ネガティブフィードバック | -10000〜0  |
| 通報                     | -20000〜0  |

## 2. ユーザー信頼性スコア (Tweepcred)

- PageRankベースの信頼性算出
- フォロー/フォロワー比率で調整
- フォロー数 > 2500 の場合、比率が悪いとペナルティ

## 3. コンテンツ最適化

### 推奨

- 画像/動画を添付（滞在時間向上）
- 質問形式で会話を誘発
- 450文字前後の説明文（X Premium長文機能活用）
- 投稿頻度: 1日1〜3回（質重視）
- 投稿時間: 7-9時、12-13時、20-22時
- 句読点は使わない（改行で区切る）
- 絵文字は🔥🚀✅☝️👇👉👈のみ

### 避けるべき

- 外部リンク単体投稿（滞在時間減少）
- ハッシュタグ乱用（スパム判定リスク）
- 攻撃的・批判的な内容
- 3日以上の投稿空白

## 4. 実装での活用

- ポスト生成スキルでこのガイドを参照
- チェックリストとして活用可能
