## コンテンツガイドライン

### トーン・スタイル

| 要素 | 推奨                     | 避ける       |
| ---- | ------------------------ | ------------ |
| 文体 | カジュアル・親しみやすい | 堅い・形式的 |
| 視点 | 実践・経験ベース         | 理論・抽象的 |
| 姿勢 | 学習中・成長中           | 上から目線   |

### 絵文字使用

**推奨:**

- 🚀 成長・発展・スピード
- 🔥 熱量・モチベーション
- ✅ 完了・達成・チェック
- ☝️ ポイント強調
- 👇 下記参照・詳細案内

**使用ルール:**

- 1投稿に2-3個程度
- 冒頭または末尾に配置
- 内容と関連性のある絵文字のみ

### 文章構造

**推奨形式:**

```
[フック/タイトル]🔥

[本文ポイント1]
[本文ポイント2]
[本文ポイント3]

[CTA/質問]👇
```

**ルール:**

- 句読点は使わない（改行で区切る）
- 1文は短く（40文字以内推奨）
- 箇条書きで読みやすく
