X(Twitter)アルゴリズムに最適化されたポスト作成のための詳細ガイド。
ソース: github.com/twitter/the-algorithm + 2026年最新情報

| ------------------- | ------ | ------------------------------------- | -------------- |

| 信号                     | ペナルティ | 回避方法                 |

## 評価カテゴリ詳細

### 1. リプライ誘発力（30点満点）

**加点要素:**

- 質問形式で終わる: +20
- 意見を求める表現: +15
- 「あなたは？」「どう思う？」: +10
- 選択肢を提示: +10

**減点要素:**

- 一方的な断言: -10
- クローズドな表現: -5

**例:**

```
✅ 良い例:
「Claude Codeでの開発効率が3倍になった🚀
みなさんはどんなワークフローで使ってる？」

❌ 悪い例:
「Claude Codeは最高のツールです」
```

### 2. エンゲージメント力（25点満点）

**加点要素:**

- 画像/動画添付: +15
- 具体的な数値・成果: +10
- 実践的なTips: +10
- ユニークな視点: +5

**減点要素:**

- 外部リンク単体: -15
- 宣伝色が強い: -10
- 一般的すぎる内容: -5

### 3. 滞在時間（25点満点）

**加点要素:**

- 適切な文字数（100-280文字）: +10
- 箇条書き/構造化: +5
- コードスニペット: +5
- 画像/動画: +10
- スレッド形式: +5

**減点要素:**

- 短すぎる（50文字未満）: -10
- 長すぎる（外部リンク誘導）: -5
- 読みにくい構造: -5

### 4. コンテンツ品質（20点満点）

**加点要素:**

- 情報価値が高い: +15
- オリジナリティ: +10
- 読みやすさ: +5
- 専門性: +5

**減点要素:**

- 既出情報の焼き直し: -10
- 誤情報: -20
- 曖昧な内容: -5

## コンテンツ最適化

### 推奨事項

1. **画像/動画を添付**
   - 滞在時間が平均2倍に増加
   - スクリーンショット、図解が効果的

2. **質問形式で会話を誘発**
   - 「〜と思いませんか？」
   - 「みなさんの経験は？」
   - 「どう思いますか？」

3. **適切な文字数**
   - 推奨: 100-280文字
   - X Premium長文: 450文字前後も可

4. **投稿タイミング**
   - 朝: 7-9時
   - 昼: 12-13時
   - 夜: 20-22時

5. **投稿頻度**
   - 1日1〜3回（質重視）
   - 3日以上の空白は避ける

### 避けるべき事項

1. **外部リンク単体投稿**
   - 滞在時間が減少
   - 必ずコンテキストを追加

2. **ハッシュタグ乱用**
   - 2個以下に制限
   - スパム判定リスク

3. **攻撃的・批判的な内容**
   - ネガティブフィードバック誘発
   - アカウント評価低下

4. **句読点の使用**
   - 改行で区切る
   - 視認性向上

## 推奨絵文字

使用推奨:

- 🔥 熱量・盛り上がり
- 🚀 成長・発展
- ✅ 完了・確認
- ☝️ ポイント強調
- 👇 下記参照
- 👉👈 方向指示

避けるべき:

- 過度な絵文字（3個以上連続）
- 内容と無関係な絵文字

## 修正優先度

1. **外部リンク単体の解消**（-15点回避）
2. **質問形式の追加**（+20点獲得）
3. **文字数調整**（+10点獲得）
4. **箇条書き追加**（+5点獲得）
