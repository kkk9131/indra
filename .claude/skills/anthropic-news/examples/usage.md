# 使用例

## 基本的な使用

```
/anthropic-news
```

両方のソース（ドキュメント・ブログ）から情報を取得。

## ドキュメントのみ

```
/anthropic-news docs
```

Claude Code公式ドキュメントから最新情報を取得。

## ブログのみ

```
/anthropic-news blog
```

Anthropicブログから最新記事を取得。

## 自然言語での発動

```
claude最新情報を教えて
```

```
anthropic newsを確認して
```
