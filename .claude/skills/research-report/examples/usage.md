# Research Report - 使用例

## 例: AIエージェント市場調査

```
ユーザー: 「2026年のAIエージェント市場について調べて」

1. [Phase 1] トピック分析
   - 検索クエリ設計:
     - "AIエージェント 市場規模 2026"
     - "AI agent market trends 2026"
     - "AIエージェント 企業 導入事例"
     - "autonomous AI agents comparison"

2. [Phase 2] WebSearchで情報収集
   - 各クエリで検索実行
   - 信頼性の高いソースを優先的に記録

3. [Phase 3] 情報整理
   - 市場規模データの整理
   - 主要プレイヤーの特定
   - トレンドの分析

4. [Phase 4] レポート作成
   - 構造化されたMarkdownレポートを作成

5. [Phase 5] 保存・納品
   - agent-output/research-20261208-ai-agent-market/report.md
```

## 例: 技術比較調査

```
ユーザー: 「ReactとVueの比較レポートを作成して」

検索クエリ例:
- "React vs Vue 2026 comparison"
- "React Vue パフォーマンス比較"
- "React Vue 学習コスト"
- "React Vue 企業採用事例"

出力先:
- agent-output/research-20261208-react-vue-comparison/report.md
```

## 例: URLからのリサーチ

```
ユーザー: 「このURLの内容についてリサーチして https://example.com/article」

1. WebFetchでURLの内容を取得
2. 関連するキーワードで追加検索
3. 情報を統合してレポート作成
```
